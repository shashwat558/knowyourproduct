# KnowYourProduct  

**KnowYourProduct** is an AI-powered tool that helps users find similar products or get a concise summary of any product from Myntra. Whether you're looking for alternatives or need a quick overview, this tool provides AI-driven insights to enhance your shopping experience.  

## Features  

- 🔍 **Find Similar Products** – Get AI-suggested alternatives for any product from Myntra.  
- 📝 **Product Summary** – Receive a concise AI-generated summary of a product’s details.  
- ⚡ **Quick & Efficient** – Saves time by providing instant recommendations and insights.
